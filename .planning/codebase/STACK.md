# Stack Overview

## Frontend
- Framework: React 18
- Build Tool: Vite
- Styling: TailwindCSS
- Routing: React Router DOM
- Icons: Lucide React
- Realtime: Socket.IO Client

## Backend
- Runtime: Node.js
- Framework: Express.js
- Database: MongoDB via Mongoose
- Object Storage: Cloudinary
- Realtime: Socket.IO
- Validation: Zod
