# Conventions

## Naming
- Components: PascalCase `MyComponent.jsx`
- Hooks: camelCase starting with use `useAuth.js`
- Variables/Functions: camelCase
- Assets: kebab-case

## Styling Conventions (Frontend)
- Adheres to Neo-Brutalism but actively transitioning to "Futuristic Dark Minimalism with Glassmorphism" per latest UI instruction.
- Uses Tailwind CSS utility classes heavily.
- Custom `@layer` directives built in `index.css`.
