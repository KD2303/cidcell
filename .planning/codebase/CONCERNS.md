# Concerns

- **Styling Overhaul**: Active radical UI redesign is underway, moving from Neo-Brutalism defaults in Tailwind config to "Futuristic Dark Minimalism". This could cause severe visual regressions if not carefully managed component-by-component.
- **Backend Coupling**: Making sure UI changes do not require breaking data requirements or validation constraints from the backend to remain seamless.
- **Role Isolation**: Role-specific layouts (`/admin`, `/mentor`, etc.) might take significant effort to style uniformly under the new dark minimal aesthetic.
